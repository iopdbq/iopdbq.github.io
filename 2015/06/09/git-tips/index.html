git tips

Git note

(1)init a repo

$mkdir proj
$cd proj
$git init    #init proj (repo) in ~

(2)commit to local-db

$touch <filename>             #make a file
$vi <filename>                  #edit a file
$git status                      #check repo status
$git add <filename>         #add to 暂存区(index)
$git status
$git commit -m "message"    #commit to local-db(HEAD)
$git status
$git log                    #commit log

注意：if出现

warning: LF will be replaced by CRLF in
原因：回车(CR, ASCII 13, \r) ，换行(LF, ASCII 10, \n)
在Windows中广泛使用CRLF（回车换行）标识一行的结束
在Linux/UNIX系统中只有LF（换行符）

  • 解决办法
    $rm -rf .git #delte .git files

    $git config —global core.autocrlf false #disable autocrlf

    $git init #reinit ~ repo

(3)continue

  • go to
    (2)commit to local-db

(4)tips

  • gitk: built-in GUI
  • undo last commit
    $git commit -m ‘cc’
    $git add forgoten_file
    $git commit —amend #modify last commit
  • recovery a file (history version)
    $git log filename #get history version num
    $git reset num
