Sublime editor intro usages

Sublime text 2 入门

Basic usage

（一）快捷键 (shortcut keys)

`Ctrl-Shift-P`             #Tools -> Command Palette(命令面板)
`Ctrl-P`                 #File -> Switch tabbed-page(file)
`Ctrl-R/(-P-@)`                     #Func -> Switch to function
`Ctrl-P-#`                         #Find -> Search in file
`Ctrl-G/(-P-:[num])`             #Line -> Go to Line num
`Ctrl-click`                     #Cursor -> Multi Selection
`Ctrl-<Shift>-/`                #Line -> Comment Line
`Select Lines Ctrl-Shift-L`     #Line -> Multi Selection
`[Obj] Ctrl-D/Alt-F3`             #Cursor -> Multi Selection
`hold Mouse-Middle-Key`            #block -> Multi Selection
`Ctrl-;`                        #字段    -> go to 字段
`Ctrl-\` `                        #CLI  ->  Python CLI
`Alt-Shift-[num]`                #Screen -> Split Screen
`Ctrl-I/<Shift>-F/H`            #bottom -> Find & Match 替换
`Ctrl-L`                        #Line ->  Select whole Lines
`Ctrl-l-Ctrl-Shift-L`            #Cursor -> Lines endings
`Ctrl-<Shift>-m`                #Cursor -> 括号内起始or结束
`Shift-(right-Select`            #block -> Multi Selection
`Ctrl-Shift-Space`                #block -> Multi Selection
`Ctrl-Shift-[/]`                #Code -> 折叠与展开
`Fn-(<-/->（方向键）`            #Line -> go to 行首/行尾
`Ctrl-kk`                        #Edit -> del cursor to 行尾
`Ctrl-k-Shift/BackSpace`        #Edit -> del cursor to 行首
`Ctrl-Shift-up/down（方向键）   #Line -> cursor swap 上下行
`Ctrl-Shift-d`                    #Line -> cp Line & insert 
`Ctrl-Enter`                    #Line -> insert Line 上一行
`Ctrl-Shift-Enter`                #Line -> insert Line 下一行
`Ctrl-[/]`                        #Line -> add/sub indenting
`Shift-Tab`                        #Line -> add/sub indenting
`Ctrl-u/z`                        #Edit -> Undo <operations>
`Ctrl-y`                        #Edit -> Recover undo
`Ctrl-BackSpace`                #Edit -> del word
`Ctrl-kl/ku`                    #Word -> Switch Up/Low word
`Ctrl-x`                        #Line -> cut whole Line
`Ctrl-t`                        #Word -> cursor swap 左右词
`Ctrl-Shift-v`                    #Edit -> paste & format
`Ctrl-<Shift>-Tab`                #File -> Switch file(order)
`Alt-[num]`                        #File -> Switch file(number)
`Ctrl-F2`                        #Mark -> Add/del flag
`Ctrl-j`                        #Line -> Join Selected Lines
